# 🏠 InspectPro PWA - แอพตรวจบ้านทำงานออฟไลน์ 100%

แอปพลิเคชันจัดการข้อมูลการตรวจรับบ้านสไตล์มินิมอล โทนสีขาว-แดง ออกแบบให้สามารถติดตั้งลงในมือถือเป็น Progressive Web App (PWA) และทำงานได้โดยไม่ต้องใช้อินเทอร์เน็ตหน้างาน พร้อมระบบเก็บบันทึกข้อมูลผ่าน LocalStorage ของเครื่องเบราว์เซอร์

## 🚀 ฟังก์ชันการใช้งาน
1. **เพิ่มโครงการได้ไม่จำกัด**
2. **ในแต่ละโครงการ สามารถเพิ่มแปลง / เลขที่บ้านได้**
3. **ในรายแปลง สามารถเพิ่มใบงานตรวจได้** โดยเลือกได้ว่าเป็น:
   - ใบงานตรวจบ้าน Stock (ซ่อนช่องเซ็นของลูกค้าอัตโนมัติ)
   - ใบงานตรวจบ้านก่อนโอนกับลูกค้า (มีช่องเซ็นลายมือชื่อร่วมกัน)
4. **เพิ่มรายการจุดบกพร่อง (Defect) ได้เรื่อยๆ ในหนึ่งใบงาน**
5. **ถ่ายรูปหน้างามผ่านกล้องมือถือ** หรืออัปโหลดภาพ พร้อมเขียนบันทึกอธิบาย
6. **ระบบเซ็นลายเซ็นอิเล็กทรอนิกส์ดิจิทัล** ทั้งฝั่ง Site Engineer (SE) และลูกค้า 
7. **ส่งออกรายงานเป็นไฟล์ PDF ทันที** พร้อมจัดหน้ารูปภาพและลายเซ็นอย่างสวยงาม

## 🛠️ วิธีการเอาไป Deploy บน GitHub Pages ทันที

เนื่องจากนี่คือแอปประเภท Static Web App (HTML/CSS/JS) คุณสามารถเปิดใช้ฟังก์ชัน **GitHub Pages** เพื่อใช้งานแอปพลิเคชันได้ฟรีใน 3 ขั้นตอน:

1. **แตกไฟล์ ZIP นี้** ออกมาเป็นโฟลเดอร์โปรเจกต์
2. **สร้าง Repository ใหม่บน GitHub** ของคุณ แล้วอัปโหลดไฟล์ทั้งหมดขึ้นไป (`index.html`, `style.css`, `app.js`, `manifest.json`, `sw.js` และโฟลเดอร์ `libs/`)
3. **เปิดใช้งาน GitHub Pages**:
   - ไปที่แท็บ **Settings** ของ Repository บน GitHub
   - เลื่อนหาเมนู **Pages** ด้านซ้ายมือ
   - ที่หัวข้อ **Build and deployment** -> เลือก Source เป็น `Deploy from a branch`
   - ที่หัวข้อ **Branch** -> เลือกเป็น `main` (หรือ `master`) และโฟลเดอร์ `/ (root)` แล้วกด **Save**
4. รอประมาณ 1-2 นาที GitHub จะให้ URL ลิงก์เว็บของคุณ (เช่น `https://username.github.io/repository-name/`) สามารถเปิดใช้งานผ่านมือถือได้ทันที!

## 📱 วิธีติดตั้งเป็นแอพลงบนมือถือ (PWA)
- **iOS (Safari):** เปิดลิงก์ URL -> กดปุ่ม **Share (แชร์)** ด้านล่าง -> เลือก **"Add to Home Screen" (เพิ่มไปยังหน้าจอโฮม)**
- **Android (Chrome):** เปิดลิงก์ URL -> จะมีป๊อปอัพขึ้นชวนติดตั้ง หรือกดที่จุด 3 จุดมุมขวาบน -> เลือก **"Install App" (ติดตั้งแอป)**
